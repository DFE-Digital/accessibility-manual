Using autocomplete attributes

The autocomplete attribute provides information to the browser, and assistive technologies, about the type of information expected in the field. This helps users to complete forms more quickly and accurately.

Example:

When a user starts typing their first name, the browser will suggest the user's first name if it has been saved in the browser previously.

You can find a list of all the list of values for autocomplete attributes on the W3C website.

The following example is a field which asks the user to provide a telephone number.

The code includes autocomplete="tel" which tells the browser that the field is expecting a telephone number.

<div class="govuk-form-group">
    <label class="govuk-label" for="phone-number">
        What is your phone number?
    </label>
    <input class="govuk-input" id="phone-number" name="phone-number" autocomplete="tel">
    type="text" 
</div>

WCAG criteria

This feature is also required to meet 1.3.5 - Identify Input Purpose (AA).

The following concepts from the knowldge base are important for interaction designers to understand.

Assistive technologies

Assistive technologies are software or hardware that help people, including those with disabilities use technology. Examples include screen readers, magnifiers, and voice recognition software.

Markup and HTML

HTML is the language used to create web pages. It is important to use semantic HTML to ensure that content is accessible to people using assistive technologies.

Roles, states and properties

Roles, states and properties are used to describe the behaviour of elements on a web page. They are important for making sure that content is accessible to people using assistive technologies.

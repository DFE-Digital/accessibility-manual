Many DfE services contain similar accessibility issues. We update the issues list below, as we identify changes in accessibility issues in our products and services.

Top issues in DfE

Find out more about each issue, and how to avoid them:

  • error messages
  • images, charts, and alt text
  • non-descriptive links
  • use of colour
  • incorrect heading structure
  • status changes to content not provided to screen readers

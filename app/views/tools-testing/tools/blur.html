Blur

Simulates blurred vision to test visibility of key content.

Use to test

How to use

Drag the links to your bookmark bar:

  • Light blur
  • Medium blur
  • Heavy blur

Once the blur tools are saved, visit a webpage and select the one you want to try in your bookmark bar.

Example of using blur

The blur tool will blur the content on the page. This simulates someone with cataracts or poor vision.

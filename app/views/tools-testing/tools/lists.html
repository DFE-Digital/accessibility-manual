Impairments this tool can test for:

  • Vision

How to use

Drag the view lists link to your bookmark bar.

Once the tool is installed, visit a webpage and select the tool in your bookmark bar.

When you have run the check, the following list types, and the type of list, will be highlighted.

Types of lists:

  • <ol> ordered list
  • <ul> unordered list
  • <dl> definition list

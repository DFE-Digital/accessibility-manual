Criteria level

This is a level AA criteria. You must meet this criteria in the Department for Education.

Goal

Ensure that the purpose of each input field that collects user information is understandable by assistive technologies and browsers through the use of the autocomplete attribute.

How this benefits users

This criteria has several benefits for everyone, specifially:

  • it helps people complete forms more quickly and accurately
  • it enables people to more easily understand the purpose of each input field
  • it means people with memory problems don't have to struggle to remember certain information
  • icons can be provided by assisitive technology based on the attribute which can help people with motor-skills dificulties

How to Implement

Identify form fields that collect personal information (e.g., name, email, address) that are used in your form.

Check the list of values for autocomplete attributes on the W3C website.

Add the autocomplete attribute to the input with the appropriate value to provide a hint to the browser about the type of information expected in the field.

Additional guidance

Read the guidance on W3C for this criteria.
